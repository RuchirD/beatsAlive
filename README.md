BeatsAlive
==========
# HTML5 Mp3 Player 
![alt text](http://gauravbehere.freehostia.com/BeatsAlive/Images/bg.png)
Based on HTML5 Audio API and D3 for audio visualization 

# Features
- Playlist
- Equliizer
- Audio Visualization
- Audio Controls: Play, Pause, Stop, Next, Previous, Seek, Current Progress
